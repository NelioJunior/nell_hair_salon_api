Nesta interação um cliente realiza uma operação de agendamento dentro de um estabelecimento de beleza. 
Preencha um json a partir das intenções demostradas pelo cliente a partir do que é dito dentro da frase.
A estrutura do json devera ter o formato descrito a seguir:

chave 'servicos', preenchido com um array com o nome dos serviços que o cliente demonstrou desejar, 
       isso não inclui profissionais do estabelecimento nem data e hora de reservas.
chave 'intencao', Preenchido seguindo a regra a seguir:
    'incluirReserva': O cliente expressou o desejo de fazer uma nova reserva de serviço de beleza.
    'incluirReserva': O cliente expressou a preferencia a horario, serviço de beleza ou funcionário especialista em beleza.
    'incluirReserva': O cliente expressou o desejo de precisar de um servico de um profissional especialista em beleza. 
    'concordancia': O cliente expressou sinal de aprovação, por exemplo dizendo sim.
    'discordancia': O cliente expressou discordância, por exemplo dizendo não.
    'alterarReserva': O cliente expressou o desejo de fazer uma mudança em uma reserva anteriormente efetuada.
    'cancelarReservaJaEfetuada': O cliente expressou o desejo de cancelar uma reserva anteriormente efetuada.
    'cancelarOperacaoEmAndamento': O cliente expressou o desejo de desistir da operação que está realizando.
    'listarReservaUsuario': O cliente expressou o desejo de se informar sobre as reservas feitas por ele.
    'listarHorariosLivres': O cliente expressou o desejo de saber os horários livres do estabelecimento em um determinado período.
    'listarFuncionarios': O cliente expressou o desejo de agendar com algum funcionário específico do estabelecimento.
    'listarFuncionarios': O cliente expressou o desejo de saber quando algum funcionário esta disponivel no estabelecimento.
    'listarFuncionarios': O cliente expressou o desejo de saber qual funcionário é especialista em algum tipo de serviço do estabelecimento.
    'listarEspecialidades': O cliente expressou o desejo de obter informações sobre os serviços realizados pelos funcionários do estabelecimento.
    'infoEmpresa': O cliente expressou o desejo de saber sobre disponibilidade do estabelecimento.
    'infoEmpresa': O cliente expressou desejo de saber informações sobre a empresa, como por exemplo, horário de funcionamento e endereço.
    'semrelacao': O cliente expressou gratidão. 
    'semrelacao': Resposta do cliente sem tem relação com os itens anteriores.
    'semrelacao': O cliente expressou comprimentos ou saudação.
