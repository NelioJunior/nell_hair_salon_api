Nesta interação um cliente realiza uma operação de agendamento dentro de um estabelecimento de um Spa para serviços de saúde, bem estar e beleza. 
Preencha um json a partir das intenções demostradas pelo cliente a partir do que é dito dentro da frase.
A estrutura do json devera ter o formato descrito a seguir:

chave 'servicos', preenchido com um array com o nome dos serviços que o cliente demonstrou desejar, 
       isso não inclui profissionais do estabelecimento nem data e hora de reservas.
       
chave 'intencao', Preenchido seguindo a regra a seguir:
    'incluirReserva' para o caso em que cliente expressou o desejo de fazer uma nova reserva de serviço de beleza.
    'incluirReserva' para o caso em que cliente expressou a preferencia a horario, serviço de beleza ou funcionário especialista em beleza.
    'incluirReserva' para o caso em que cliente expressou o desejo de precisar de um serviço de um profissional especialista em beleza. 
    'incluirReserva' para o caso em que cliente citou um nome de uma pessoa que nao seja ela mesmo,sinal que provalmente quer agendar um serviço com a pessoa citada.
    'incluirReserva' para o caso de o cliente citou uma hora ou data aleatoria,sinal que provalmente o cliente esteja indicando a data ou horario da reserva.
    'concordancia' para o caso em que cliente expressou sinal de aprovação, por exemplo dizendo sim.
    'discordancia' para o caso em que cliente expressou discordância, por exemplo dizendo não.
    'discordancia' para o caso em que cliente expreessou que acredita que ocorreu algum erro.  
    'indiferenca' para o caso em que cliente expressou indiferença a qualquer opção oferecida.
    'alterarReservaJaEfetuada' para o caso em que cliente expressou o desejo de fazer uma mudança em uma reserva anteriormente efetuada.
    'cancelarReservaJaEfetuada' para o caso em que cliente expressou o desejo de cancelar uma reserva anteriormente efetuada.
    'cancelarReservaJaEfetuada' para o caso em que cliente comunicou que nao vai poder vir ou ir ao estabelecimento.
    'cancelarOperacaoEmAndamento' para o caso em que cliente expressou o desejo de desistir da operação que está realizando.
    'listarReservaUsuario' para o caso em que cliente expressou o desejo de se informar sobre as reservas feitas por ele.
    'listarHorariosLivres' para o caso em que cliente expressou o desejo de saber os horários livres do estabelecimento em um determinado período.
    'listarFuncionarios' para o caso em que cliente expressou o desejo de agendar com algum funcionário específico do estabelecimento.
    'listarFuncionarios' para o caso em que cliente expressou o desejo de saber quando algum funcionário esta disponivel no estabelecimento.
    'listarFuncionarios' para o caso em que cliente expressou o desejo de saber qual funcionário é especialista em algum tipo de serviço do estabelecimento.
    'listarEspecialidades' para o caso em que cliente expressou o desejo de obter informações sobre os serviços realizados pelos funcionários do estabelecimento.
    'infoEmpresa' para o caso em que cliente expressou o desejo de saber sobre disponibilidade do estabelecimento.
    'infoEmpresa' para o caso em que cliente expressou desejo de saber informações sobre a empresa, como por exemplo, horário de funcionamento e endereço.
    'gratidao' para o caso em que cliente expressou gratidão. 
    'semrelacao'= Resposta do cliente sem tem relação com os itens anteriores.
    'semrelacao' para o caso em que cliente expressou comprimentos ou saudação. 
    'semrelacao' para o caso em que cliente expressou satisfação por ter conseguido agendar.
